# Data Visualization

A data visualization dashboard for analyzing and presenting data.

## 🎯 Features

- Interactive charts and graphs
- Multiple visualization types (bar, line, pie, scatter)
- Data import/export
- Customizable dashboards
- Real-time data updates
- Responsive design

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/data-viz.git
cd data-viz

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- Plotly/Chart.js
- Pandas
- See `requirements.txt` for full list
